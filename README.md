# Forine.org - Checker (Tools)
Forine.org (или же [Forine.net](http://forine.net)) это один из многочисленных серверов с мини-играми в minecraft. С каждым годом нарушителей находить становится все сложнее и сложнее, и поэтому в команде проекта начались обсуждения "Что же с ними делать?".

Пришли к тому, что нужно создать универсальный чекер для проверки игроков в читерстве. В этом репозитории находится сборка инсрументов для проверки "читера" от одного из администраторов проекта - Velvi. 

Версия чекера (или же как назвал автор VelviCheck3000): **2.1**
